Detailed Description

Class representing a map of binary pixels.

Used to create an adjacency list representation of a graph of pathways between adjacent pixels.

Attention
Zeros indicate the fields while ones indicate the walls.
Examples
create_list_from_pixelmap.cpp, and create_pixel_map.cpp.
