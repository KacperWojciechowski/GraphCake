An example of adding an edge between two given vertices in Graph::Matrix object.This example shows how to add a new edge to the graph structure in the Graph::Matrix object. The function does verify whether given indexes are correct, and the given weight is non-zero. In case the edge already exists, this function modifies the weight the edge, as shown in the different example linked below. Depending on the graph type, this function inserts an edge or modifies its weight either one-way in case of directed graph, or both ways in case of undirected graph.
