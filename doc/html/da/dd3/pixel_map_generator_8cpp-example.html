An example of generating a randomized pixel map data source via Data::Generator class.This example shows how to generate a randomized pixel map data. The output is directed to an output stream passed as an argument to the function, and as such, can redirect the output to file or to standard output.

Note
To redirect the function result to the standard output, pass std::cout reference as an argument.
